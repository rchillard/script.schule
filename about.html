What is Script Schule?

There's a thousand resources in the world to learn JavaScript, but not many that provide you interactive tools to really retain it.

Script Schule features a set of tools you can use to brush up on your modern JavaScript by:

  • Building your memory with our interactive trainer
  • Solving code problems in different domains
  • Coding along by typing ahead of a solution
